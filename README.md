# CurrencyConversion
A currency conversion app

## Introduction
The CurrencyConversion is a demo for check exchange currency rate. The Repo is using androidx and for libs use as latest as possible.

## Third party libs
Third-party be used in this repo:

- Rxjava2
- OkHttp
- retrofit
- Glide
- Dagger2
- Data Binding

Test libs:

- Power mock
- Mockito
